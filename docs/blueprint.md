# **App Name**: GhostTalk

## Core Features:

- Anonymous Access: Enable users to access chats instantly without login or email requirements.
- End-to-End Encryption: Implement end-to-end encrypted (E2EE) peer-to-peer messaging for secure communications.
- Automatic Message Deletion: Automatically delete messages after a configurable time period to enhance privacy.
- Chat Room Pairing: Provide options for random chat room pairing or private invite link generation for controlled access.
- Local Message Encryption: Locally encrypt all messages before transmission to ensure data protection.
- Metadata Stripping Tool: Use a tool that removes all identifiable metadata from user messages to enhance anonymity.

## Style Guidelines:

- Primary color: Dark purple (#4A148C) for a mysterious and secure feel.
- Background color: Very dark gray (#212121) for a clean, modern, dark-mode-only appearance.
- Accent color: Electric purple (#BE29EC) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Use minimalist icons to represent different chat functions and settings. Icons should be monochromatic with the accent color.
- A clean, uncluttered layout to ensure ease of use and focus on privacy features. Utilize negative space effectively.
- Subtle fade-in and fade-out animations for messages and transitions to give a smooth user experience.
